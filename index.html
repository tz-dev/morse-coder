Help

Enter Morse code with the main button, left mouse button, or Space.

Press briefly for a dot. Hold longer for a dash. The Morse tone plays while the input is held and stops on release.

The left bar shows the pause window. In Game Mode it turns red after a completed letter and counts down to Game Over.

The right bar shows input feedback in Training Mode. In Game Mode it shows your progress through the target word.

Difficulty changes the letter pause, auto-finish timing, game-over timeout, and input delay for short and long signals.

In Game Mode, type the target word correctly. A wrong signal or too long a pause ends the round.
